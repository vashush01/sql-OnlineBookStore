# sql-OnlineBookStore
# 📚 SQL Book Store Management Project

A beginner-friendly SQL project built using PostgreSQL to practice database creation, data manipulation, and query writing. This project simulates a simple **Book Store Management System**, where various SQL queries are used to retrieve meaningful business insights from the database.

---

## 📌 Project Overview

This project demonstrates fundamental SQL concepts by creating a relational database for a bookstore and performing different analytical queries on the data.

The project covers:
- Database Creation
- Table Creation
- Data Insertion
- Data Retrieval
- Filtering Records
- Sorting Data
- Aggregate Functions
- Business-Oriented SQL Queries

---

## 🛠️ Tech Stack

- PostgreSQL
- pgAdmin 4
- SQL
- Visual Studio Code

---

## 📂 Database Structure

The project contains the following tables:

### 📖 Books
- Book ID
- Title
- Author
- Genre
- Price
- Published Year
- Stock

### 👤 Customers
- Customer ID
- Customer Name
- Email
- Country

### 🛒 Orders
- Order ID
- Customer ID
- Book ID
- Order Date
- Quantity
- Total Amount

---

## 🔍 SQL Queries Performed

Some of the SQL operations included in this project are:

- Retrieve books by genre
- Find books published after a specific year
- Retrieve customers from a specific country
- Display orders within a date range
- Calculate total stock available
- Find the most expensive book
- Filter customers based on order quantity
- Retrieve high-value orders
- List unique genres
- Find books with the lowest stock
- Sorting using ORDER BY
- Filtering using WHERE
- Aggregate Functions (SUM, COUNT, AVG)
- DISTINCT
- LIMIT

---

## 📊 Sample Business Questions Solved

- Which books were published after 1950?

- What orders were placed during November 2023?
- What is the total stock available?

- Which orders have a total amount greater than $20?

5. Run the queries to view the results.
